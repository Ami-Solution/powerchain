# PowerChain
This is an early drop for Smart Contracts (Solidity/Ethereum) to operate a Micro-Smart-Grid. 

## Entities

### Termination
Grid Operator (DSO)

### Metering
Smart Meter Operator (Asset Owner)

### Meter
Smart Meter 

### Node
Grid Node (Customer) 

### PoweDelivery
Power Delivery Contract
